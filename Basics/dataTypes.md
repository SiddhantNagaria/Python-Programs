# Object Types/ Data Types

- Numbers: 1234, 3.13 , 3 + 4j, 0b111, Decimal(), Fraction()
- Strings: "Hello, World!", 'Hello, World!', """Hello, World!""",
- Boolean: True, False
- List: [1, 2, 3], ["a", "b", "c]
- Tuple: (1, 2, 3), ("a", "b", "c)
- Dictionary: {"name": "John", "age": 30}, {"key": "value}
- Set: {1, 2, 3}, {"a", "b", "c}
- NoneType: None
